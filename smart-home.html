Smart Home

Semsa includes a Smart Home tool for controlling supported Shelly Wi-Fi devices. You can manage lights manually from the widget or use voice control inside the app.

How to connect See screenshots Back to app page

What is supported

  • Only Shelly products with Wi-Fi support
  • No Bluetooth-only devices yet
  • Manual widget control
  • Voice control through Semsa tools

Quick summary

Put the Shelly device into AP / Wi-Fi setup mode, connect it to your home Wi-Fi in the app, save it, then control it from the widget or by voice.

How to connect a Shelly device

Use this flow when adding a new light or other Shelly device to Semsa.

1

Put the device into Wi-Fi setup mode

Switch the Shelly device off and on twice, so it enters the API / Wi-Fi mode. In this mode it starts its setup network and becomes discoverable.

2

Find the device in Semsa

Open Smart Home in the app and search for the device while it is broadcasting its setup Wi-Fi.

3

Enter your home Wi-Fi

Choose your home Wi-Fi network and type the password so the Shelly device can join your network.

4

Save and control it

Save the device and then use the widget or voice control to turn lights on and off, adjust brightness, and run smart home actions.

At the moment we support only Shelly devices with Wi-Fi capability. Bluetooth-only devices are not included yet.
